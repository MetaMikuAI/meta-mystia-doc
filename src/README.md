# 项目介绍

[**MetaMystia**](https://github.com/MetaMikuAI/MetaMystia)是一个为游戏《[东方夜雀食堂](https://store.steampowered.com/app/1584090/)》开发的**Mod**（玩家自制扩展）。

如果您不熟悉“Mod”这个词，可以简单理解为：

> 一个安装在原版游戏上的“补丁式插件”，用来给游戏增加原本没有的新功能。

它的目标很简单：

> 让原本只能单人游玩的夜雀食堂，真正“热闹”起来。

通过MetaMystia，您可以与朋友一同经营食堂、同步行动与操作，共同应对顾客需求，将原本的单机体验扩展为可以一起玩的联机模式。

## 这是什么？

《东方夜雀食堂》原生并不支持联机游玩。MetaMystia通过Mod的方式，为游戏引入了**双人联机能力**。

如果用更直白的话说：

- 原版游戏：只能一个人玩
- 安装MetaMystia后：可以和朋友一起玩

您可以将它理解为：

- 一个让夜雀食堂“支持一起游玩”的功能扩展
- 一个还在不断更新、变得更稳定、更好用的联机系统

本项目由[**MetaMiku**](https://github.com/MetaMikuAI)与[**ShigureYuki**](https://github.com/ShigureYuki)共同开发和维护。

## 已实现的核心功能

目前，MetaMystia已经完成了较为完整的联机基础，覆盖了绝大多数实际游玩场景。

### 联机与同步基础

- 基础联机功能
- 跨地图状态同步
- 较为自然的角色移动同步
- 游戏内Console（一个简单的“调试窗口”，主要给开发者或高级玩家查看运行状态用，普通玩家可以不用理会）
- …

### 经营内容同步

在联机过程中，以下内容都能够在玩家之间保持一致：

- 菜单状态同步
- 酒水状态同步
- 厨具状态同步
- 预设状态同步
- “开始营业”状态同步
- …

### 顾客相关行为同步

MetaMystia对顾客系统进行了较完整的同步支持，包括：

- 顾客生成同步
- 顾客入座同步
- 顾客点单同步
- 上菜过程同步
- 顾客离开同步
- …

这些同步共同构成了一个完整、可游玩的联机经营流程，而不仅仅是“能看到彼此”。

## 其他额外内容

### 新的稀有客人

MetaMystia计划为游戏添加8位新的稀客，包括：

- 大妖精
- 小恶魔
- 芙兰朵露
- 八意永琳
- 神绮
- 秋静叶
- 秋穰子
- 八云蓝

这些稀客有着精心撰写的角色设定与各自独特的喜好、厌恶倾向，希望您玩得开心。

> [!NOTE]
> 已完成7位，其余仍在制作中。

### 新的羁绊任务、料理、酒水等

MetaMystia将为新增的8位稀客编写专属的羁绊任务和剧情，带来新的料理与酒水以丰富游戏体验。

> [!NOTE]
> 这部分内容仍在开发中，如果您对制作内容或测试感兴趣，也欢迎参与。

## 仍在持续完善中

除了上面列出的功能，MetaMystia仍在不断改进和扩展，包括：

- 提高联机稳定性
- 解决联机时可能出现的剧情冲突问题
- 支持更多原版游戏系统的联机玩法
- 支持玩家自定义新的稀客、料理和食材等
- 为以后新增功能预留扩展空间
- …

这是一个**正在成长的联机Mod**，而不是已经完美的作品。
