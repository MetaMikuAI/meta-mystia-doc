# 项目介绍

[**MetaMystia**](https://github.com/MetaMikuAI/MetaMystia)是一个为游戏《[东方夜雀食堂](https://store.steampowered.com/app/1584090/)》开发的**联机Mod**。它的目标很简单：

> 让原本只能单人游玩的夜雀食堂，真正“热闹”起来。

通过MetaMystia，您可以与朋友一同经营食堂、同步行动与操作，共同应对顾客需求，将原本的单机体验扩展为协作式的联机玩法。

## 这是什么？

《东方夜雀食堂》原生并不支持联机游玩。MetaMystia通过Mod的方式，为游戏引入了**双人联机能力**。

您可以将它理解为：

- 一个让夜雀食堂“支持双人”的功能扩展
- 一个面向玩家与开发者持续演进的联机框架

本项目由[**MetaMiku**](https://github.com/MetaMikuAI)与[**ShigureYuki**](https://github.com/ShigureYuki)共同开发和维护。

## 已实现的核心功能

目前，MetaMystia已经完成了较为完整的联机基础，覆盖了绝大多数实际游玩场景。

### 联机与同步基础

- 实现基础联机功能
- 支持跨地图状态同步
- 实现较为稳定、自然的角色运动同步
- 提供游戏内Console，用于调试与状态查看
- …

### 经营内容同步

在联机过程中，以下内容都能够在玩家之间保持一致：

- 菜单状态同步
- 酒水状态同步
- 厨具状态同步
- 预设状态同步
- “开始营业”状态同步
- …

### 顾客相关行为同步

MetaMystia对顾客系统进行了细致的同步支持，包括：

- 顾客生成同步
- 顾客入座同步
- 顾客点单同步
- 上菜过程同步
- 顾客离开同步
- …

这些同步共同构成了一个完整、可游玩的联机经营流程，而不仅仅是“能看到彼此”。

## 其他额外内容

### 新的稀有客人

MetaMystia计划为游戏添加8位新的稀客，包括：

- 大妖精
- 小恶魔
- 芙兰朵露
- …

这些稀客有着精心撰写的角色设定与各自独特的喜好、厌恶倾向，希望您玩得开心。

> [!NOTE]
> 已完成5位，其余仍在制作中。

### 新的羁绊任务、料理、酒水等

MetaMystia将为新增的8位稀客编写专属的羁绊任务和剧情，带来新的料理与酒水以丰富游戏体验。

> [!NOTE]
> 这部分的内容和实现方式仍在开发中，欢迎感兴趣的各位一同参与。

## 仍在持续完善中

除了上面列出的功能，MetaMystia仍在不断扩展和优化中，包括：

- 同步稳定性的持续提升
- 更多游戏系统的联机支持
- 自定义角色小人
- 解决剧情冲突问题
- 为后续功能和扩展预留接口
- …

这是一个**正在成长的联机Mod**，而不是已经完美的作品。
