# 常见问题（FAQ）

如果您遇到的问题未在以下问题中列出，请查看[获取帮助](./how_to_use.md#get_help)。

## 通用问题

### 只想体验新内容，不想联机可以吗？

MetaMystia所提供的额外内容（如：新的稀客、料理和食材等）**不强制要求**联机，您可阅读[下载安装Mod](./how_to_install.md#install_mod)获取更多信息。

## 联机相关问题

### 联机需要新开一个存档吗？

不需要新开一个存档，联机**可以随时开始**，不要求存档进度一致。

### 一方已经结束夜间营业，另一方却无法结束，怎么办？

在联机过程中，如果**一名玩家已经结束了夜间营业**，而**另一名玩家卡住、无法正常结束**，这是已知的偶发现象。

可以让无法结束的一方，在控制台中输入以下命令进行强制结束：

```bash
/call try_close_izakaya
```

执行后通常可以正常结束当晚营业并继续流程。

### 可以一起过剧情吗？

可以**一起体验一部分剧情**，但目前**并不是所有剧情都能稳定联机触发**。

如果遇到以下情况：

- 剧情无法继续
- 剧情触发异常

建议的处理方式是：

- **先断开联机**
- 各自单机完成当天流程或剧情
- 第二天再重新联机继续游玩

目前剧情冲突在开发阶段**是难以完全避免的**。
如果遇到具体问题，欢迎在群内反馈，并**附上日志文件**，有助于后续修复。

### 一方有DLC，另一方没有或DLC不全，怎么办？

MetaMystia已对DLC不一致的情况做了基础兼容处理：

- **准备阶段**
    - 禁止选择**对方缺少的DLC角色、料理、酒水等资源**

- **夜间营业阶段**
    - 禁止取出对方缺少的DLC料理或酒水
    - 禁止制作对方缺少的DLC料理

### 白天采集时间是怎么处理的？

联机状态下：

- **白天时间是各自独立进行的**
- 每位玩家可以按自己的节奏采集、跑图、处理事务
- 游戏会在**晚上六点前等待双方都准备完成**

也就是说，不会互相抢时间，也不会影响对方的白天操作。

### 雇佣的伙伴是怎么处理的？

联机期间伙伴会**自动禁用**，联机断开后会自动解禁。

### 联机期间没有时停，夜雀之歌相关的Buff还在吗？

- **夜雀之歌**
    - 谁做菜触发了夜雀之歌，就**由谁自己演唱**
    - 不会强制同步给另一位玩家
- **抛菜**
    - 目前**仍然保留**
    - 可正常使用
