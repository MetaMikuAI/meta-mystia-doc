# 常见问题（FAQ）

如果您遇到的问题未在以下问题中列出，请查看[获取帮助](./how_to_use.md#get_help)。

## 通用问题

### 只想体验新内容，不想联机可以吗？

可以。MetaMystia所提供的额外内容（如：新的稀客、料理和食材等）**不强制要求**联机，您完全可以把它当成一个“内容扩展Mod”，单人游玩时同样可以正常体验这些新东西。

您可以阅读[下载安装Mod](./how_to_install.md#install_mod)获取更多信息。

### 为什么有时候游戏窗口会卡死、白屏，并有弹窗提示“该程序未响应”？

这个现象可能是因为您意外地用鼠标点到了随游戏启动一起弹出的黑色控制台窗口，在该窗口内的交互（如：选中文本）会导致游戏进程被挂起。

您可以尝试在黑色控制台窗口内点击一下**鼠标右键**以取消选中文本，观察游戏窗口是否恢复正常。

### 如何卸载MetaMystia？

#### 一键卸载

我们提供了便捷管理MetaMystia Mod的工具，您可以阅读[下载安装Mod](./how_to_install.md#onclick_install)获取更多信息。

#### 手动卸载

如果您**只希望**移除MetaMystia相关文件，删除`游戏根目录/BepInEx/plugins/MetaMystia-v\*.dll`文件与`游戏根目录/ResourceEx`文件夹即可。

如果您还希望移除和Mod有关的所有文件，换言之，将游戏还原为不支持Mod的状态，您**还需要**删除以下文件和文件夹（如有）：

- `游戏根目录/BepInEx`
- `游戏根目录/.doorstop_version`
- `游戏根目录/changelog.txt`
- `游戏根目录/doorstop_config.ini`
- `游戏根目录/MinHook.x64.dll`
- `游戏根目录/winhttp.dll`

## 联机相关问题

### 联机需要新开一个存档吗？

不需要。您和朋友都可以继续使用自己原本的存档，联机**可以随时开始**，不要求存档进度一致。

- 每个人的存档是独立的
- 剧情进度、地图解锁、角色解锁、材料数量都不会互相共享

联机时的规则可以简单理解为：

> 白天各玩各的，晚上一起开店。

更具体来说：

- 白天：各自跑图、采集、做任务，互不影响
- 晚上营业：用料由“做菜的人”自己消耗，营业赚到的钱双方共享

### 一方已经结束夜间营业，另一方却无法结束，怎么办？

在联机过程中，如果**一名玩家已经结束了夜间营业**，而**另一名玩家卡住、无法正常结束**，这是已知的偶发现象。

可以让无法结束的一方，在控制台中输入以下命令强制结束当晚营业：

```bash
/call try_close_izakaya
```

执行后通常可以正常结束当晚营业并继续流程。

### 可以一起过剧情吗？

可以一起体验**一部分剧情**，但目前**并不是所有剧情都能稳定联机触发**。

如果遇到以下情况：

- 剧情无法继续
- 剧情触发异常

建议的处理方式是：

- **先断开联机**
- 各自单机完成当天流程或剧情
- 第二天再重新联机继续游玩

目前剧情冲突在开发阶段**是难以完全避免的**。
如果遇到具体问题，欢迎在群内反馈，并**附上日志文件**，有助于后续修复。

### 一方有DLC，另一方没有或DLC不全，怎么办？

MetaMystia已对DLC不一致的情况做了基础兼容处理：

- **准备阶段**
    - 禁止选择**对方缺少的DLC角色、料理、酒水等资源**

- **夜间营业阶段**
    - 禁止取出对方缺少的DLC料理或酒水
    - 禁止制作对方缺少的DLC料理

### 白天采集时间是怎么处理的？

联机状态下：

- **白天时间是各自独立进行的**
- 每位玩家可以按自己的节奏采集、跑图、处理事务
- 游戏会在**晚上18点前等待双方都准备完成**

也就是说，不会互相抢时间，也不会影响对方的白天操作。只有两个人都进入“准备完成”状态，才会一起进入夜间营业。

### 雇佣的伙伴还会帮忙吗？

联机期间伙伴会**自动禁用**，联机断开后会自动解禁。这样做是为了避免AI和真人玩家之间的操作冲突。

### 联机期间没有时停，夜雀之歌相关的Buff还在吗？

- **夜雀之歌**
    - 谁做菜触发了夜雀之歌，就**由谁自己演唱**
    - 不会强制同步给另一位玩家
- **抛菜**
    - 目前**仍然保留**
    - 可正常使用
